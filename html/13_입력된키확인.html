- keydown 이벤트
      -> key가 내려갈 때, 내려가 있을 때 포함

    - keyup 이벤트
      -> key가 올라왔을 때, 1회 발생

    - 요소.addEventListener("keydown", function(e){

      // 매개변수 e : 이벤트 객체
      // 현재 수행된 이벤트와 관련된 모든 정보를 가지고 있음

      // e.key : 어떤 key가 down/up 되었는지 반환해줌
    })

    - document.addEventListener("keydown", function(e){})
